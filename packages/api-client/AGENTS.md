# @workspace/api-client

Type-safe API client package generated from an OpenAPI specification.
It uses `openapi-fetch` and `openapi-typescript` to fully type API requests and responses.

## Purpose

- Single entry point for `apps/web` to call the API server (`apps/api`).
- Provides type safety via generated types from the OpenAPI spec (`schema.d.ts`).
- Includes utilities for error handling.

## File layout

```
src/
  client.ts     # Factory function `createApiClient`
  error.ts      # `ApiError` type and `throwOnError` utility
  index.ts      # Public exports
  openapi.json  # OpenAPI spec fetched from the API server (committed)
  schema.d.ts   # Types generated by `openapi-typescript` (committed)
scripts/
  fetch-openapi-spec.ts   # Fetch `openapi.json` from the API server
  generate-api-types.ts   # Generate types: `openapi.json` → `schema.d.ts`
```

## Scripts

```bash
# Fetch the OpenAPI spec from the API server (http://localhost:3010)
bun run api:fetch

# Generate types from the local openapi.json (no API server needed)
bun run api:generate

# Fetch the spec and generate types (two-step)
bun run api:fetch && bun run api:generate

# Run TypeScript type check
bun run typecheck
```

## Type update workflow

When the API spec changes:

1. Run the `apps/api` server (`bun dev`).
2. Run `bun run api:fetch` to update `src/openapi.json`.
3. Run `bun run api:generate` to regenerate `src/schema.d.ts`.
4. Commit both files.

## Usage

```ts
import { createApiClient, throwOnError } from "@workspace/api-client"

const client = createApiClient({ baseUrl: "http://localhost:3010" })

// throwOnError throws on error and returns data on success
const data = throwOnError(
  await client.GET("/essays/{id}", { params: { path: { id } } })
)
```

## Error handling

`ApiError` extends `Error` and includes the following fields:

- `status`: HTTP status code
- `code`: `"unauthorized"` (401) | `"remote_api_error"` (other)

```ts
import { throwOnError, type ApiError } from "@workspace/api-client"

try {
  const data = throwOnError(await client.GET("/essays"))
} catch (e) {
  const error = e as ApiError
  if (error.code === "unauthorized") {
    /* ... */
  }
}
```

## Notes

- `schema.d.ts` and `openapi.json` are generated files — do not edit them manually.
- This package is intended to be used only by `apps/web`. Importing it from `apps/api` would create a circular dependency.
- `credentials: "include"` is configured by default to support cookie-based authentication.
