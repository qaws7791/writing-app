## 현재 상태

저장소에는 빌드, 린트, 타입 검사, Storybook 빌드 스크립트가 존재합니다. 배포 파이프라인과 환경별 정책은 아직 기준 문서가 필요한 단계입니다.

## 최소 릴리스 절차

1. 변경 범위 확인
2. `bun lint`, `bun typecheck`, 필요한 경우 `bun build` 실행
3. 관련 문서 갱신 여부 확인
4. UI 변경 시 Storybook 또는 시각 검증
5. 배포 후 핵심 흐름 확인

## 추후 구체화할 항목

- 환경별 배포 대상
- 마이그레이션 절차
- 롤백 기준
- 비밀정보 관리 방식
