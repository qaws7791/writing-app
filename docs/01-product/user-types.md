# User Types

## 1. 시작이 어려운 입문자

- 글을 쓰고 싶지만 주제 선정과 첫 문장에서 자주 멈춘다.
- 정답보다 작은 시작 신호가 필요하다.

## 2. 루틴을 만들고 싶은 생활형 작성자

- 매일 조금씩 쓰고 싶지만 꾸준함이 유지되지 않는다.
- 오늘 무엇을 하면 되는지 빠르게 알고 싶다.

## 3. 자기 목소리를 지키고 싶은 작성자

- AI 도움은 받고 싶지만 자기 문체를 잃고 싶지 않다.
- 전체 대필이 아니라 표현과 연결성 점검을 원한다.

## 4. 공개가 두려운 작성자

- 피드백은 필요하지만 평가와 오해가 두렵다.
- 공개 범위와 피드백 채널을 조절할 수 있어야 한다.
