## 핵심 전제

글필은 에세이 '생산' 도구가 아니라 에세이 '훈련' 플랫폼입니다.
이 문서는 여정·세션·스텝 구조의 설계 철학적 근거를 정의합니다.

## 커리큘럼 원칙

이론 전달보다 실제 작성 행동을 늘리는 쪽에 초점을 둡니다.

**원칙:**

- 짧고 반복 가능한 단위로 훈련한다 (세션 단위, 20~30분)
- 설명보다 예시와 직접 수행을 우선한다 (스텝별 실행 → 피드백)
- 초고 작성과 퇴고를 별도 기술로 다룬다
- 사용자의 주제와 경험을 기반으로 훈련한다 (글감 기반 글쓰기)
- 정답형 평가보다 표현력과 구조화 능력의 향상을 본다

**다루는 핵심 역량:**

| 역량             | 관련 스텝 유형              |
| ---------------- | --------------------------- |
| 글감 발견        | WRITING_PROMPT 탐색         |
| 논지 세우기      | CONCEPT, MULTIPLE_CHOICE    |
| 문단 구성        | ORDERING, FILL_IN_THE_BLANK |
| 사례와 근거 배치 | WRITING, EXAMPLE            |
| 문장 명료화      | REWRITING, AI_COMPARISON    |
| 퇴고와 회고      | AI_FEEDBACK, COMPLETION     |

## 학습 루프

사용자가 항상 다음 행동을 알 수 있도록 구조화된 흐름입니다.

```
발견 → 준비 → 작성 → 다듬기 → 회고 → (공유)
```

| 단계   | 글필 구현                       | AI 개입        |
| ------ | ------------------------------- | -------------- |
| 발견   | 글감 탐색, 여정 선택            | ✗              |
| 준비   | INTRO, CONCEPT, EXAMPLE 스텝    | ✗              |
| 작성   | WRITING, SHORT_ANSWER 스텝      | 제한적         |
| 다듬기 | REWRITING, AI_COMPARISON 스텝   | AI 피드백 제공 |
| 회고   | COMPLETION 스텝, 세션 완료 화면 | AI 격려 메시지 |
| 공유   | (Phase 2 이후)                  | —              |

**설계 의도:**

- 작성과 학습을 분리하지 않고 같은 흐름(여정) 안에 둔다
- AI는 준비와 다듬기 단계에서만 제한적으로 개입한다

## 활동 체계

세션 내 스텝은 다음 활동 유형 원칙을 따릅니다:

| 활동 유형 | 예시 스텝                   | 설계 규칙             |
| --------- | --------------------------- | --------------------- |
| 워밍업    | CONCEPT, HIGHLIGHT          | 5~15분                |
| 구조 훈련 | ORDERING, FILL_IN_THE_BLANK | 결과가 초고로 이어짐  |
| 표현 훈련 | REWRITING                   | 모호→명료화           |
| 퇴고 훈련 | AI_COMPARISON               | 삭제·재배열·중복 제거 |
| 회고 훈련 | COMPLETION                  | 막힌 지점 기록        |

**설계 규칙:**

- 한 스텝은 5~15분 안에 끝날 수 있어야 한다
- 결과는 다음 스텝 또는 자유 글쓰기 초안으로 이어져야 한다
- 성취보다 반복과 전이를 우선한다

## 평가 전략

점수화보다 사용자가 자신의 성장 방향을 이해하도록 돕는 것이 목적입니다.

**성장 지표 (진행률에 반영):**

- 세션 시작 빈도: 얼마나 자주 여정을 시작했는가
- 완성 흐름: 첫 스텝에서 COMPLETION까지 이어지는가
- 구조화: 문단과 논지 전개가 나아지는가 (AI_FEEDBACK 기반)
- 표현 명료성: 모호함과 중복이 줄어드는가
- 회고 습관: 세션 완료 후 스스로 진단하는가

**피해야 할 것:**

- 한 번의 결과물(글)만 보고 사용자를 평가하는 방식
- AI가 생성한 점수에 과도하게 의존하는 방식
- 연속접속일(스트릭)을 글쓰기 성과처럼 사용하는 방식
