## 1. 온보딩 및 첫 여정

처음 앱을 실행한 사용자가 관심사와 수준을 설정하고, 여정 중심의 경험을 시작한다.

1. 앱 최초 실행 → 온보딩 화면 표시
2. 관심 분야 선택 (예: 마음챙김) + 글쓰기 경험 선택 (예: 입문)
3. 홈 화면에 개인화된 추천 여정 카드가 최상단에 표시됨
4. 회원가입 없이 글쓰기 탭에서 체험 글쓰기 가능
5. "저장" 탭 시 회원가입 유도 ("가입하면 글이 저장됩니다")

## 2. 여정 기반 구조화된 학습

사용자가 여정을 선택하고 세션과 스텝을 순서대로 진행하며 글쓰기 역량을 쌓는다.

### 2-1. 여정 탐색 및 시작

1. 여정 탭 또는 홈 화면에서 여정 탐색
2. 카테고리 필터 칩(전체 / 글쓰기 기술 / 마음챙김 / 실용 글쓰기)으로 탐색
3. 여정 카드 탭 → 여정 상세(설명, 전체 진행률, 세션 리스트) 확인
4. "지금 시작하기" 탭 → 세션의 첫 번째 스텝이 전체 화면으로 시작

### 2-2. 홈에서 여정 이어하기

1. 홈 화면에 진행 중인 여정 카드가 최상단에 표시됨
2. "이어하기 →" 버튼 탭 → 해당 여정의 현재 세션 첫 번째 스텝으로 바로 이동

### 2-3. 스텝별 흐름

| 스텝 유형       | 사용자 행동                                     | 시스템 반응                                                    |
| --------------- | ----------------------------------------------- | -------------------------------------------------------------- |
| **개념 학습**   | 설명 읽고 "다음" 탭                             | 다음 스텝으로 전환, 프로그레스 바 업데이트                     |
| **가이드 질문** | 100자 이상 답변 입력 후 "답변 제출"             | 답변 저장 후 다음 스텝 진행                                    |
| **글쓰기**      | 글 작성 후 "제출"                               | 로딩("글을 분석하고 있습니다") → 10초 이내 AI 피드백 스텝 전환 |
| **AI 피드백**   | 강점/개선점/질문 확인 후 "수정하기" 또는 "다음" | 수정 에디터 열림 또는 다음 스텝                                |
| **수정**        | 피드백 참고해 수정 후 "재제출"                  | 초안 대비 비교 분석, 개선된 부분 하이라이트                    |

### 2-4. 세션 완료 및 다음 세션 잠금 해제

1. 마지막 스텝 완료 → 세션 완료 애니메이션 표시
2. 해당 세션 상태가 "완료"로 변경
3. 다음 세션이 잠금 해제되어 "진행 중"으로 전환
4. 여정 상세의 진행률 업데이트

### 2-5. 여정 완료 후 글쓰기 연결

1. 여정의 마지막 세션 완료 → 여정 완료 축하 화면 표시
2. "글쓰기 공간에서 배운 내용을 자유롭게 표현해보세요" CTA 표시
3. CTA 탭 → 글쓰기 탭의 글쓰기 진입 화면으로 이동

### 2-6. 세션 중 앱 종료 후 재진입

1. 세션 진행 중 앱이 예기치 않게 종료됨
2. 재실행 후 해당 세션으로 진입하면 마지막으로 완료한 스텝부터 재개
3. 이전에 입력한 내용이 보존되어 있음

## 3. 글쓰기 (글감 포함)

사용자가 글쓰기 탭에서 자유 글쓰기를 시작한다. 글감은 글쓰기 진입 화면과 에디터 내 Bottom Sheet에서 보조적으로 제공된다.

### 3-1. 글쓰기 진입 (글감 미리보기)

1. 글쓰기 탭 → "새 글 쓰기" 버튼 탭
2. "오늘 어떤 글을 써볼까요?" 진입 화면 표시
3. 필터 칩(전체 / 감각 / 회고 / 의견)으로 글감 탐색
4. 글감 카드에 유형 태그, 제목, 요약, 응답 수("N명이 이 글감으로 글을 썼어요") 표시
5. 글감 카드 탭 → 해당 글감이 참조 카드로 고정된 에디터 열림
6. "직접 쓸게요" 버튼 탭 → 빈 에디터 열림

### 3-2. 에디터 내 글감 추가 탐색

1. 빈 에디터에서 본문이 비어 있는 동안 "아이디어가 필요하신가요?" 버튼 표시
2. 버튼 탭 → 글감 선택 Bottom Sheet 표시
3. Bottom Sheet에서 필터 칩(전체 / 감각 / 회고 / 의견) + 글감 카드 리스트
4. 글감 카드 탭 → Bottom Sheet 닫힘 → 에디터 상단에 글감 참조 카드 고정
5. "직접 쓸게요" 버튼 탭 → Bottom Sheet 닫힘, 빈 에디터로 복귀

### 3-3. 글 작성 및 저장

1. 글 작성 중 5초마다 자동 저장 (별도 알림 없음)
2. 완료(✓) 탭 → 서재에 저장, 글 상세(읽기 뷰)로 전환
3. 본문 입력 시 "아이디어가 필요하신가요?" 버튼 자동 숨김

## 4. 서재 관리

사용자가 작성한 모든 글을 글쓰기 탭의 서재에서 조회하고 관리한다.

1. 글쓰기 탭 기본 화면에서 글 목록 (최신순)
2. 검색 바에 키워드 입력 → 제목/본문에서 키워드 검색
3. 글 카드 탭 → 글 상세(읽기 뷰)
4. 글감을 사용한 글: "WRITING PROMPT" 접힘 카드 탭 → 글감 제목·본문 펼쳐짐
5. 글감 없이 작성된 글: Writing Prompt 카드 미표시
