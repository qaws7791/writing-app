## 시대적 맥락

숏폼 콘텐츠와 알고리즘 기반 소비 문화는 주의 집중 시간을 단축하고, 비판적 사고보다 감정 반응을 우선하는 미디어 소비 습관을 형성했다. 글쓰기의 사회적 필요가 감소하면서(편지, 일기, 보고서 → 메신저, 이모지, 음성 메시지), 논리적 구성 능력과 표현력이 전반적으로 저하되고 있다. 생성형 AI 도구의 남용은 "모름 → 질문 → AI가 즉답"이라는 단축 경로를 제공하여 스스로 사유하는 과정 자체를 건너뛰게 만든다.

## 사용자가 겪는 구체적 고통

| 고통 영역           | 핵심 문제                                                  | 결과                     |
| ------------------- | ---------------------------------------------------------- | ------------------------ |
| **아이디어 생성**   | 무엇을 써야 할지 막막함, 소재/주제 생성 어려움             | 글쓰기 시작 자체를 회피  |
| **구성·구조화**     | 서론-본론-결론의 전체 틀과 문단 구성에 대한 불안           | 비구조적이고 산만한 글   |
| **완벽주의·미루기** | 초고를 "형편없다"고 판단하여 반복 갈아엎음                 | 품질/만족도 저하, 미완성 |
| **퇴고 어려움**     | 초고-퇴고-버전 관리가 어렵고 "나중에 고치기"가 무한 대기열 | 성장 없는 반복           |
| **학습 결핍**       | 에세이 구조, 퇴고 전략에 대한 체계적 학습 기회 부재        | 실력 정체                |
| **공개 불안**       | "내 글이 틀릴까", 타인의 평가에 대한 자기검열              | 글쓰기 중단, 침묵 효과   |
| **AI 의존**         | AI 생성 텍스트의 획일화, 개성 약화, 사고력 퇴화            | 비판적 사고 근육 상실    |

## 기존 도구의 한계

### 생산성 앱의 한계

수집가의 오류(Collector's Fallacy): 기사, 링크, 문장 조각을 캡처하고 분류하는 행위 자체를 "사고했다"고 착각하게 만든다. 정보의 단순한 축적은 통찰로 이어지지 않는다. 진정한 사고력 향상은 정보를 분석하고, 기존 지식과 연결하며, 결론을 도출하는 과정에서 발생한다.

### 모바일 앱의 한계

- **구조적 사고 방해:** 한 번에 보이는 정보량이 적어 전체 맥락 파악이 어렵고, 글을 국소적으로만 다루게 된다.
- **집중 분산:** 앱 알림과 메신저로 인해 글쓰기 흐름(flow state)이 자주 끊긴다.
- **비선형적 사고 지원 부족:** 메모 → 순서 변경 → 묶기 → 나누기 같은 복합적 사고 과정보다 위에서 아래로 입력하는 방식에만 최적화되어 있다.

### 생성형 AI의 한계

- **허위·환각:** AI는 확신에 찬 틀린 정보를 생산하고, 존재하지 않는 출처를 인용하기도 한다.
- **문체 획일화:** 가장 일반적인 표현을 선택하므로, 여러 사용자의 글이 비슷한 어투와 구조로 수렴된다.
- **저자의 개성 약화:** 문법적으로 뛰어나지만 "개인적인 목소리와 경험, 독특한 시각"이 부족한 텍스트를 생성한다.
- **스킬 저하:** 장기적으로 스스로 쓰는 능력 습득을 저해하고, 비판적 사고 근육을 약화시킨다.
- **"AI 티 나는 글"에 대한 거부감:** 결과물이 지나치게 매끈하고 일반적이어서, 오히려 글쓴이의 문제의식이 흐려진다.
