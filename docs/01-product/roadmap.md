## MVP (P0) — 핵심 루프 구축

글감 선택 → 글쓰기 → AI 피드백의 핵심 3단계 루프를 완성한다. 여정 기반 구조화된 학습의 기본 골격을 제공하는 것이 목표다.

**인증 및 온보딩**

- Google/Kakao 소셜 로그인 (Better Auth)
- 온보딩: 관심사 및 글쓰기 수준 선택 → 개인화 글감·여정 추천

**홈**

- 시간대별 동적 인사 및 오늘의 추천 글감 카드
- 진행 중인 여정 카드 (진행률 프로그레스 바)
- 홈 / 글감 / 여정 탭 전환

**글감 (Writing Prompt)**

- 글감 카드 리스트 (감각 / 회고 / 의견 필터)
- 글감 상세 (본문, 응답 수)
- 글감에서 에디터 바로 연결

**여정 (Journey)**

- 여정 카테고리 (글쓰기 기술 / 마음챙김 / 기술)
- 여정 상세 및 세션 리스트 (선형 진행, 잠금 해제)
- 세션 스텝 플로우 (개념 학습 → 가이드 질문 → 글쓰기 → AI 피드백 → 수정)
- 세션 완료 축하 애니메이션 및 다음 세션 잠금 해제

**글 에디터 & 서재**

- 클린 에디터 (실시간 단어 수 카운터, 5초 자동 저장)
- 서재(Library): 글 목록 조회, 검색
- 글 상세(읽기 뷰): Writing Prompt 연결 카드

**AI 피드백**

- 강점 / 개선점 / 사고 촉발 질문 3단 피드백 (10초 이내)
- 수정본 재제출 시 초안 대비 비교 분석
- 수준별 피드백 비율 조정 (입문 2:1, 중급 1:1, 상급 1:2)

**게이미피케이션**

- 여정 진행률 (프로그레스 바, 달성률 %)

## Phase 1 — 리텐션 강화

사용자가 꾸준히 돌아오는 동기를 강화한다.

- **글쓰기 스트릭:** 연속 글쓰기 일수 추적 및 시각화
- **단어 수 누적:** 총 작성 단어 수 카운터
- **사고력 성장 대시보드:** 세션 완료 수, 글 작성 수, 수준 변화 시각화
- **프로필 통계:** 완료한 여정 수, 작성한 글 수
- **알림:** 스트릭 유지 알림, 새 글감 알림
- **오프라인 지원:** 서재 조회 및 에디터 작성, 온라인 복귀 시 자동 동기화

---

## Phase 2 — 심화 경험

- **여정 완료 뱃지:** 프로필에 뱃지 추가
- **데이터 내보내기:** 모든 글을 마크다운 파일로 다운로드
- **접근성:** 글꼴 크기 조절 (보통/크게/매우 크게), 동작 줄이기 모드
- **구독 관리:** Free / Pro 구독 상태 확인 및 변경 (Pro: 전체 여정 + 고급 AI 코칭)
- **북마크:** 글감 북마크 저장 및 조회
