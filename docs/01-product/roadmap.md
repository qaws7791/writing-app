## Phase 1 — MVP (0~3개월)

> **목표:** 여정 핵심 루프 검증

**인증 및 온보딩**

- Google/Kakao 소셜 로그인 (Better Auth)
- 온보딩: 관심사 및 글쓰기 수준 선택 → 개인화 여정 추천

**홈**

- 시간대별 동적 인사
- 여정 진행 현황 대시보드 (진행 중인 여정 카드 최우선 배치)
- 진행 중인 여정이 없을 시 "첫 여정 시작하기" CTA
- 하단에 "오늘 자유롭게 글을 써보세요" 보조 카드

**여정 (Journey) — Core**

- 여정 탭: 카테고리별 여정 탐색(글쓰기 기술 / 마음챙김 / 실용 글쓰기), 여정 상세
- 세션 진행 플로우 (전체 스텝 유형): Learn / Read / Guided Question / Write / Feedback / Revise
- 세션 완료 축하 애니메이션 및 다음 세션 잠금 해제
- 여정 완료 시 글쓰기 탭으로 자연스러운 연결 CTA

**글쓰기 — Secondary**

- 글쓰기 탭: 서재(글 목록) + "새 글 쓰기" 버튼
- 글쓰기 진입 화면("오늘 어떤 글을 써볼까요?") — 글감 미리보기 + "직접 쓸게요" 옵션
- 클린 에디터: 실시간 단어 수 카운터, 5초 자동 저장
- 글감 참조 카드(선택적): 에디터 상단 접힘/펼침
- "아이디어가 필요하신가요?" 버튼 → 글감 선택 Bottom Sheet
- 글 상세(읽기 뷰): Writing Prompt 접힘 카드 (글감 상세 보기 링크 없음)

**글감 (Writing Prompt) — Tertiary**

- 글감 카드 (유형 태그 + 제목 + 요약 + 응답 수 소형 표시)
- 글쓰기 진입 화면 + 에디터 내 Bottom Sheet에서만 노출
- 독립 글감 탭/상세 화면 없음

**AI 피드백**

- 강점 / 개선점 / 사고 촉발 질문 3단 피드백 (소크라테스식, 수준별)
- 수정본 재제출 시 초안 대비 비교 분석

**게이미피케이션**

- 여정 진행률 (프로그레스 바, 달성률 %)
- 세션 완료 애니메이션

## Phase 2 — Growth (3~6개월)

> **목표:** 리텐션 강화 + 콘텐츠 확장

- 여정 콘텐츠 확대 (카테고리별 3개 이상)
- 글쓰기 스트릭 + 단어 수 누적 게이미피케이션
- 사고력 성장 대시보드: 세션 완료 수, 글 작성 수, 수준 변화 시각화
- 프로필 통계: 완료한 여정 수(우선), 작성한 글 수
- 오프라인 지원 + 자동 동기화
- 개인화 강화 (AI 기반 여정 추천)
- Pro 구독 결제 (인앱 결제)

---

## Phase 3 — 심화 경험

- 여정 완료 뱃지: 프로필에 뱃지 추가
- 데이터 내보내기: 모든 글을 마크다운 파일로 다운로드
- 접근성: 글꼴 크기 조절 (보통/크게/매우 크게), 동작 줄이기 모드
- 구독 관리: Free / Pro 구독 상태 확인 및 변경
- 알림: 스트릭 유지 알림, 새 여정 알림
