## Phase 1. 쓰기 시작 만들기

- 홈에서 오늘의 추천 행동 제시
- 글감 탐색과 저장
- 모바일 우선 에디터
- 초고 저장과 기본 퇴고 흐름

## Phase 2. 학습 루프 강화

- 짧은 글쓰기 훈련 활동
- 구조와 전개를 돕는 스캐폴딩
- 회고와 자기 피드백 화면

## Phase 3. AI 보조 고도화

- 선택 영역 교정
- 전체 문서 검토
- 문단 연결과 표현 중복 피드백

## Phase 4. 안전한 공개와 운영

- 공개 범위 제어
- 피드백 정책과 모더레이션
- 운영 도구와 지원 프로세스
