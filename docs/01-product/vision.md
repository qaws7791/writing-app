## 제품 비전

> **"여정을 통해 성장하고, 글쓰기로 표현하라."**

글필(Geulpil)은 한국어 사용자를 위한 에세이 글쓰기 학습 플랫폼이다. AI가 글을 대신 써주는 도구가 아니라, 사용자가 여정(Journey)이라는 구조화된 커리큘럼을 통해 단계별로 성장하고, 글쓰기 공간에서 배운 내용을 자유롭게 표현하는 학습 플랫폼이다.

## 핵심 가치 제안

| 기존 경험                       | 글필의 제안                                 |
| ------------------------------- | ------------------------------------------- |
| 무엇을 배워야 할지 모름         | 여정(Journey)이 학습 방향을 제시            |
| 빈 화면 앞에서 막막함           | 글감(Writing Prompt)이 글쓰기의 시작을 도움 |
| 한 번 쓰고 끝나는 일회성 글쓰기 | 초안 → AI 피드백 → 수정의 다단계 워크플로우 |
| 혼자 쓰고 혼자 판단             | 소크라테스식 AI 코칭 + 커뮤니티 피드백      |
| 실력이 느는지 알 수 없음        | 사고력 성장 대시보드로 인지적 발달 시각화   |
| AI가 대신 써줌 → 실력 퇴화      | AI는 질문자 역할, 사고의 주체는 사용자      |

## 제품 구조

글필은 **"여정(Journey)"** 이라는 구조화된 커리큘럼을 중심으로 사용자가 단계별로 사고력과 글쓰기 역량을 성장시키도록 설계한다. 기능은 명확한 위계를 따른다.

- **[Core] 여정(Journey):** 특정 역량 향상을 목표로 한 구조화된 커리큘럼
- **세션(Session):** 여정 내 10~20분 완결형 학습 노드
- **스텝(Step):** 개념 학습 → 예시 읽기 → 가이드 질문 → 글쓰기 → AI 피드백 → 수정의 마이크로 활동
- **[Secondary] 글쓰기(Writing):** 여정의 연습 공간. 서재(Library)와 자유 글쓰기 에디터를 포함
- **[Tertiary] 글감(Writing Prompt):** 글쓰기 진입 화면과 에디터 내에서 아이디어를 제공하는 보조 카드

## 우리가 만들고 싶은 변화

- 막막해서 시작하지 못하는 사용자가 글감을 통해 첫 문장을 쓸 수 있게 한다.
- 한 번 쓰고 끝나는 글쓰기를 초안-피드백-수정의 반복 루프로 바꾼다.
- AI를 답안 생성기가 아닌 소크라테스식 코치로 사용한다.
- 사용자의 인지적 성장을 데이터로 시각화하여 동기를 지속시킨다.

## 비범위

- AI가 사용자를 대신해 글 전체를 작성하는 기능
- 저장과 수집만을 위한 생산성 앱 기능 (노션, 옵시디언 대체 불가)
- 반응 수와 팔로워 수 중심의 소셜 피드
- 지나치게 복잡한 데스크톱 전용 편집 경험
- 일기 앱 또는 감정 치유 앱 기능
- 브런치, 문피아 같은 출판·수익화 플랫폼 기능
