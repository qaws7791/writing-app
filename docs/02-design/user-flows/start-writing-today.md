## 목표

사용자가 앱을 열었을 때 무엇을 해야 할지 고민하지 않고, 오늘 바로 한 편의 글쓰기를 시작하게 만든다.

## 범위

- 포함: 홈 진입, 오늘의 추천 글감 확인, 추천안 비교, 글감 선택, 바로 쓰기 진입
- 제외: 글감 탐색 화면의 상세 필터 조작, 학습 활동 진입, AI 보조 사용

## 시작과 종료

- 시작 조건: 로그인 상태의 사용자가 홈에 진입한다.
- 종료 조건: 사용자가 선택한 글감을 바탕으로 에디터 초고 작성 상태에 진입한다.

## 주요 감정과 장벽

- 앱을 열자마자 해야 할 일이 많아 보이면 시작이 지연된다.
- 추천 글감이 너무 일반적이면 자기 이야기로 연결하기 어렵다.
- 첫 클릭 이후 다음 행동이 불분명하면 이탈 가능성이 커진다.

## 핵심 의사결정 지점

- 추천 글감 중 하나를 바로 선택할지
- 먼저 저장해둘지
- 홈의 이어서 쓰기로 방향을 바꿀지

## 흐름 요약

1. 사용자는 홈에서 오늘의 추천 글감 영역을 본다.
2. 카드의 제목, 분위기, 예상 글 길이를 빠르게 비교한다.
3. 한 개의 글감을 선택해 상세를 보거나 바로 쓰기를 선택한다.
4. 선택한 글감이 반영된 에디터로 이동해 초고를 시작한다.
