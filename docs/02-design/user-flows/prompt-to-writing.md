## 목적

글감에서 글쓰기로 이어지는 이 흐름은 글필의 가장 기본적인 사용 루프다. 소재 발견의 어려움을 해소하고 글쓰기를 즉시 시작하도록 설계한다.

## 시작 조건

- 사용자가 홈 탭 또는 글감 서브탭에 있는 상태

## 종료 조건

- 글이 서재에 저장되고 글 상세 화면에 도달한 상태

## 핵심 사용자 목표

"오늘 뭔가를 쓰고 싶은데, 주제를 찾고 바로 시작하고 싶다."

---

## 주요 흐름 (Happy Path)

```
홈 탭
  ↓
오늘의 추천 주제 카드 탭  (또는 글감 서브탭 > 글감 카드 탭)
  ↓
글감 상세 화면
  ├── 글감 제목 + 본문 읽기
  └── 참여 통계 확인 (선택적)
  ↓
"글쓰기 시작" 버튼 탭
  ↓
글 에디터
  ├── Writing Prompt 참조 카드 (접힘 상태)
  ├── 제목 입력
  ├── 본문 작성 (5초 간격 자동 저장)
  └── 완료(✓) 버튼 탭
  ↓
글 상세 화면 (읽기 뷰)
  └── 서재에 저장 완료
```

---

## 분기 흐름

### 북마크 후 나중에 쓰기

```
글감 탭 > 글감 카드 > 북마크(🔖) 탭
  ↓
글감 북마크 저장 (아이콘 상태 변경)
  ↓
나중에 재접근 → 글감 상세 → "글쓰기 시작"
```

### 비회원 사용자의 저장 시도

```
에디터 > 완료(✓) 탭
  ↓
회원가입 유도 화면
  └── "가입하면 글이 저장됩니다."
  ↓
회원가입 완료 → 글 저장 → 서재에 추가
```

### 오프라인 상태

```
에디터 > 작성 중
  ↓
로컬 자동 저장 유지
  ↓
온라인 복귀 → 자동 동기화
```

---

## 핵심 장벽

| 장벽                | 설계 해결책                                     |
| ------------------- | ----------------------------------------------- |
| 빈 화면 앞의 막막함 | Writing Prompt 참조 카드로 글감 항상 접근 가능  |
| 초고 완벽함 강박    | 자동 저장, "제출은 완성이 아닌 시작" 프레이밍   |
| 글감이 나와 안 맞음 | 유형 필터(감각/회고/의견)로 취향 맞는 글감 탐색 |

## 결정 지점

- **오늘의 추천 vs 직접 탐색:** 홈에서 추천 카드가 첫 번째 옵션, 글감 탭에서 탐색 가능
- **바로 쓰기 vs 북마크:** 글감 상세에서 CTA "글쓰기 시작"이 primary
