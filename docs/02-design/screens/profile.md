## 목적

프로필/설정 화면은 사용자가 자신의 활동 현황을 확인하고, 계정·접근성·구독을 관리하는 화면이다.

## 진입 경로

- 하단 탭 바 "프로필" 탭 탭

## 화면 구성

**페이지 타이틀**

- "설정"

**프로필 영역**

- 아바타 이미지 (원형)
- 사용자명 (예: "김수필")
- 이메일 (예: "abcd1234@email.com")

**통계 카드 (2열)**

- 완료한 여정: "여정 {N}개"
- 작성한 글: "글 {N}개"

**계정 관리 섹션**

- 계정 보안 (→ 별도 화면)
- 알림 설정 (ON/OFF 토글 또는 → 별도 화면)

**접근성 섹션**

- 글꼴 크기: "보통" / "크게" / "매우 크게" 선택
- 동작 줄이기 모드: 토글 (화면 움직임 최소화)

**서비스 및 데이터 섹션**

- 구독 관리 (Pro): 현재 구독 상태 배지 표시 (예: "ACTIVE")
- 데이터 내보내기: 탭 시 확인 다이얼로그 → 마크다운 파일 다운로드

**로그아웃**

- 빨간색 텍스트 버튼, 화면 하단

---

## 상태별 UI

### Free 구독 상태

- 구독 관리 항목: "구독 관리 (Free)" + Pro 업그레이드 CTA

### Pro 구독 상태

- 구독 관리 항목: "구독 관리 (Pro)" + "ACTIVE" 배지 + 갱신일 표시

### 데이터 내보내기 확인 다이얼로그

- "모든 글을 마크다운 파일로 내보낼까요?"
- "내보내기" (primary) / "취소" (secondary)

---

## 인터랙션

**글꼴 크기 변경**

- 변경 즉시 앱 전체에 적용 (라이브 프리뷰)

**동작 줄이기 모드**

- 활성화 시 화면 전환 애니메이션과 스크롤 효과 최소화

**알림 설정**

- 변경 즉시 저장, 알림 스케줄 업데이트
