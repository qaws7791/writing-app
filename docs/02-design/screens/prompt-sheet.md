## 목적

글감 선택 시트는 빈 에디터에서 글쓰기를 시작했지만 소재가 필요한 사용자를 위한 보조 경로다. 글쓰기 진입 화면에서 "직접 쓸게요"를 선택한 후에도 글감을 탐색할 수 있게 한다.

## 진입 경로

- 글 에디터 > 본문 비어 있는 상태 > "아이디어가 필요하신가요?" 버튼 탭

## 화면 구성

**시트 타이틀**

- "어떤 이야기를 꺼내볼까요?"

**필터 칩**

- "전체", "감각", "회고", "의견" (수평 스크롤)
- 선택 시 해당 유형 글감만 리스트에 표시

**글감 카드 리스트**

- 각 카드 구성:
  - 유형 태그 (배지): 감각 / 회고 / 의견
  - 글감 제목
  - 글감 본문 요약
  - 응답 수 소형 표시 ("N명이 이 글감으로 글을 썼어요")
- 글감 카드 탭 → 시트 닫힘 → 에디터 상단에 해당 글감이 참조 카드로 고정

**"직접 쓸게요" 버튼**

- 시트 하단에 표시
- 탭 → 시트 닫힘, 빈 에디터 상태 유지

---

## 상태별 UI

### 기본 상태

- Bottom Sheet가 화면 하반부를 가림
- 에디터 본문은 시트 뒤에 블러/딤 처리

### 글감 로딩 실패

- "글감을 불러오지 못했어요." + 재시도 버튼
- "직접 쓸게요" 버튼은 항상 사용 가능
