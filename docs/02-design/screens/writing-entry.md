## 목적

글쓰기 진입 화면은 에디터가 열리기 전 글감 탐색 기회를 제공하는 중간 단계다. 독립 글감 탭/상세 화면 없이도 글감 발견성을 확보하면서, "직접 쓸게요" 버튼으로 스킵 가능하게 하여 추가 마찰을 최소화한다.

> **도입 배경:** v2.0 리스크 평가에서 Bottom Sheet만으로는 글감 발견성이 지나치게 낮다는 점이 지적되었다. 독립 탭을 부여하는 대신, 글쓰기 시작 직전의 자연스러운 흐름 내에 글감 탐색 공간을 삽입하는 방식으로 절충한다.

## 진입 경로

- 글쓰기 탭 > "새 글 쓰기" 버튼 탭
- 여정 완료 축하 화면 > 글쓰기 CTA 탭

## 화면 구성

**화면 타이틀**

- "오늘 어떤 글을 써볼까요?"

**필터 칩**

- "전체", "감각", "회고", "의견" (수평 스크롤)
- 선택 시 해당 유형 글감만 목록에 표시

**글감 카드 목록 (3~5장)**

- 각 카드 구성:
  - 유형 태그 (배지): 감각 / 회고 / 의견
  - 글감 제목
  - 글감 본문 요약
  - 응답 수 소형 표시 ("N명이 이 글감으로 글을 썼어요")
- 글감 카드 탭 → 해당 글감이 참조 카드로 고정된 에디터가 열림

**"직접 쓸게요" 버튼**

- 화면 하단에 항상 표시
- 탭 → 빈 에디터가 열림

> **설계 의도:** 이 화면은 글감을 탐색하는 공간이지, 글감의 소셜 맥락(다른 사람의 글, 참여 통계 상세)을 보여주는 공간이 아니다. 응답 수는 심리적 안전감을 주는 사회적 증거(Social Proof) 역할에 한정한다. 글감 상세 화면은 존재하지 않는다.

## 상태별 UI

### 기본 상태

- 필터 칩 "전체" 선택 상태
- 3~5개 글감 카드가 목록으로 표시

### 글감 로딩 실패

- "글감을 불러오지 못했어요. 다시 시도해볼까요?" + 재시도 버튼
- "직접 쓸게요" 버튼은 항상 사용 가능

### 로딩 상태

- 카드 영역에 스켈레톤 표시. "직접 쓸게요" 버튼은 즉시 렌더링.
