## 목적

글 상세는 완성된 글을 읽는 경험을 제공한다. 글이 어느 글감에서 시작했는지 맥락을 확인하고, 다시 편집하거나 글감으로 이동하는 진입점 역할을 한다.

## 진입 경로

- 에디터에서 완료(✓) 탭
- 서재 > 글 카드 탭
- 글감 상세 > 작성된 글 목록 탭

## 화면 구성

**헤더**

- 뒤로 가기 (←)
- 글 제목 (중앙, 줄임)
- 더보기 메뉴 (⋮) — 편집, 삭제 등

**날짜**

- 작성일 (예: "2024년 6월 12일 수요일")

**제목**

- 큰 타이포그래피 (H1 수준)

**저자명**

- 우측 정렬 (예: "김지원")

**Writing Prompt 접힘 카드** (글감 연결 시)

- 레이블: "WRITING PROMPT" (소형 대문자)
- 글감 제목 (예: "사유의 궤적: 여백의 미학")
- 기본: 접힘(collapsed) 상태
- 탭 시 펼쳐져 글감 본문 전체 표시
- 글감 상세 보기 링크 없음 (v2.1에서 글감 상세 화면 제거)

**본문**

- 에세이 본문 전체
- 읽기 전용 (편집 불가)
- 산세리프 폰트, 적절한 줄 간격

---

## 상태별 UI

### 글감 없이 작성된 글

- Writing Prompt 접힘 카드 영역 없음

### 더보기 메뉴 (⋮)

- "편집하기" → 에디터 화면으로 이동
- "삭제" → 삭제 확인 다이얼로그 표시
