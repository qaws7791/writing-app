## 문서 역할

이 폴더는 개발 핸드오프 직전 수준의 화면 명세 문서를 관리한다.

screen 문서는 페이지 기준으로 작성하며 기본 상태, 빈 상태, 로딩 상태, 오류 상태, 주요 변형을 한 문서에서 함께 다룬다.

## 작성 기준

- 모바일을 기본 기준으로 쓴다.
- 구현 방식과 API는 적지 않는다.
- 각 문서 하단에 연결된 features와 user-flows를 명시한다.
