## 목적

나의 여정은 사용자가 현재 진행 중인 커리큘럼을 한눈에 확인하고, 새로운 여정을 탐색하는 화면이다.

## 진입 경로

- 하단 탭 바 "나의 여정" 탭 탭
- 홈 탭 > "더보기" 링크

## 화면 구성

**페이지 타이틀**

- "여정"

**진행 중인 여정 섹션**

- 현재 참여 중인 여정 카드 목록
- 각 카드:
  - 썸네일 이미지 (좌측)
  - 여정명
  - 부제 (예: "완성된 단편선", "자아 성찰 에세이")
  - 진행률 프로그레스 바 + 퍼센트
- 카드 탭 → 여정 상세 화면으로 이동

**여정 찾기 섹션**

- 섹션 타이틀: "여정 찾기" + 우측 "더보기" 링크
- 카테고리 2열 그리드
- 각 카테고리 카드:
  - 카테고리 아이콘 (예: ☀, ♡)
  - 카테고리명 (예: "일상의 발견", "관계의 밀도")
  - 포함된 테마 수 (예: "12개 테마", "8개 테마")
- 카드 탭 → 해당 카테고리의 여정 목록 또는 여정 상세

## 상태별 UI

### 진행 중인 여정 없음

- "아직 시작한 여정이 없어요. 나에게 맞는 여정을 찾아보세요!" 안내
- 여정 찾기 섹션을 더 크게 표시

### 로딩 상태

- 카드 영역에 스켈레톤 표시
