## 목적

글쓰기 탭은 사용자가 새 글을 시작하고 과거 글을 관리하는 통합 공간이다. "새 글 쓰기" 버튼으로 글감 미리보기 진입 화면에 접근하고, 서재에서 과거 글을 검색하고 다시 읽거나 편집한다.

## 진입 경로

- 하단 탭 바 "글쓰기" 탭 탭
- 에디터 > 완료(✓) 후 저장 완료
- 홈 > 글쓰기 제안 카드 탭

## 화면 구성

**페이지 타이틀**

- "글쓰기"

**[PRIMARY CTA] "새 글 쓰기" 버튼**

- 화면 상단의 가장 눈에 띄는 위치
- 탭하면 글쓰기 진입 화면([[02-design/screens/writing-entry]])으로 이동

**검색 바**

- 플레이스홀더: "기록된 생각을 검색해보세요"
- 탭 시 키보드 활성화, 글 제목+본문에서 검색

**서재 (나의 글)**

- 최신순 정렬 (기본)
- 각 글 카드:
  - 날짜 (예: "2024년 5월 12일")
  - 글 제목 (큰 폰트)
  - 본문 미리보기 (2줄)
  - 단어 수 (예: "≡ 1,240 단어")
  - 더보기 메뉴 (⋮) — 편집, 삭제

## 상태별 UI

### 빈 상태 (글 없음)

- "아직 작성된 글이 없어요.\n첫 번째 에세이를 시작해볼까요?"
- 글쓰기 진입 화면으로 이동하는 CTA 버튼

### 검색 결과 없음

- "'{검색어}'에 대한 결과가 없어요."
- 검색어 초기화 제안

### 로딩 상태

- 카드 스켈레톤 표시
