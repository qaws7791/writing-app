## 목적

서재는 사용자가 작성한 모든 글의 아카이브다. 검색으로 특정 글을 찾거나, 글 목록을 탐색하여 과거 글을 다시 읽고 편집하는 진입점이다.

## 진입 경로

- 하단 탭 바 "서재" 탭 탭
- 에디터 > 완료(✓) 후 저장 완료

## 화면 구성

**페이지 타이틀**

- "글쓰기"

**검색 바**

- 플레이스홀더: "기록된 생각을 검색해보세요"
- 탭 시 키보드 활성화, 글 제목+본문에서 검색

**글 목록**

- 최신순 정렬 (기본)
- 각 글 카드:
  - 날짜 (예: "2024년 5월 12일")
  - 글 제목 (큰 폰트)
  - 본문 미리보기 (2줄)
  - 단어 수 (예: "≡ 1,240 단어")
  - 더보기 메뉴 (⋮) — 편집, 삭제

---

## 상태별 UI

### 빈 상태 (글 없음)

- "아직 작성된 글이 없어요.\n첫 번째 에세이를 시작해볼까요?"
- 글감 탭으로 이동하는 CTA 버튼

### 검색 결과 없음

- "'{검색어}'에 대한 결과가 없어요."
- 검색어 초기화 제안

### 로딩 상태

- 카드 스켈레톤 표시
