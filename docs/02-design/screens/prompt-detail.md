## 목적

글감 상세 화면은 사용자가 글감을 깊이 이해하고, "이 글감으로 글을 쓸 것인지" 결정하는 화면이다. 다른 사람들의 응답을 통해 사회적 참여감을 제공한다.

## 진입 경로

- 홈 탭 > 오늘의 추천 주제 카드 탭
- 글감 탭 > 글감 카드 탭
- 글 상세 > Writing Prompt 접힘 카드 > "글감 상세 보기 →"

## 화면 구성

**헤더**

- 뒤로 가기 (←)
- 글감 제목 (중앙)
- 더보기 메뉴 (⋮)

**글감 제목**

- 대형 타이포그래피. 화면 상단에 두드러지게 배치.

**글감 본문**

- 사고를 촉발하는 질문 또는 상황 설명 (2~4문장)
- 읽기 쉬운 줄 간격, 적절한 여백

**참여 통계 (2열)**

- "참여 작가: {N}명"
- "작성된 수필: {N}개"

**작성된 글 리스트**

- 해당 글감에 대해 작성된 글 목록 (최신순)
- 각 항목:
  - 날짜
  - 단어 수 태그 (예: "≡ 1,240 단어")
  - 글 제목
  - 본문 미리보기 (2줄)

**CTA**

- "글쓰기 시작" — 화면 하단 고정 버튼 (primary)

---

## 상태별 UI

### 기본 상태

- 글감 제목과 본문이 화면 상단부를 차지한다.
- "글쓰기 시작" 버튼이 항상 하단에 고정된다.
- 글 목록은 스크롤로 탐색한다.

### 글 목록 없음

- "이 글감에 아직 작성된 글이 없어요. 첫 번째로 써보세요!" 안내 텍스트
- "글쓰기 시작" CTA는 동일하게 표시

### 로딩 상태

- 글감 제목과 본문 영역 스켈레톤
- CTA 버튼은 즉시 렌더링 (비활성화 상태)
