## 목적

홈은 사용자가 앱을 열었을 때 오늘 무엇을 할지 즉시 결정할 수 있는 화면이다. 3개의 수평 탭(홈/글감/여정)으로 글감 탐색과 여정 탐색을 한 곳에서 제공한다.

## 진입 경로

- 앱 최초 진입
- 하단 탭 바 "홈" 탭 탭

## 화면 구성

### 공통 영역

상단에 3개의 수평 탭이 위치한다.

```
[ 홈 ]  [ 글감 ]  [ 여정 ]
```

탭 전환은 0.3초 이내 즉시 전환한다.

---

### 홈 탭

**인사 영역**

- 시간대 기반 동적 인사: "좋은 아침입니다, {이름}님."
- 텍스트 크기가 크고, 화면 상단 좌측 배치

**오늘의 추천 글감 카드**

- 레이블: "★ 오늘의 추천 글감"
- 글감 제목 (큰 텍스트)
- 참여 통계: "지금까지 {N}명이 응답했어요"
- 카드 전체가 탭 가능 → 글감 상세 화면으로 이동

**현재 진행 중인 여정 섹션**

- 섹션 타이틀: "현재 진행 중인 여정" + 우측 "더보기" 링크 (나의 여정 탭으로 이동)
- 카드 구성:
  - 썸네일 이미지 (좌측)
  - 여정명
  - 부제 (예: "완성된 단편선", "자아 성찰 에세이")
  - 진행률 프로그레스 바 + 퍼센트 (예: "65%")
- 카드 전체 탭 → 여정 상세 화면으로 이동

---

### 글감 탭

**필터 칩**

- "전체", "감각", "회고", "의견" (수평 스크롤)
- 선택 시 해당 유형 글감만 목록에 표시

**글감 카드 리스트**

- 각 카드 구성:
  - (선택적) 브랜드/파트너 로고
  - 날짜 (YYYY년 M월 D일)
  - 유형 태그 (감각 / 회고 / 의견)
  - 글감 제목
  - 응답 수 (예: "3,892개의 수필")
  - 북마크 버튼 (🔖)
- 무한 스크롤
- 카드 탭 → 글감 상세 화면으로 이동

---

### 여정 탭

**필터 칩**

- "전체", "글쓰기 기술", "마음챙김", "기술"

**여정 카드 리스트**

- 각 카드 구성:
  - 썸네일 이미지
  - 여정명
  - 설명 텍스트
  - 세션 수 (예: "7개 세션")
- 카드 탭 → 여정 상세 화면으로 이동

---

## 상태별 UI

### 진행 중인 여정 없음

- "현재 진행 중인 여정" 섹션 대신 여정 찾기 진입 카드 표시
- 예시: "나에게 맞는 여정을 찾아보세요 →"

### 로딩 상태

- 카드 영역에 스켈레톤 표시. 인사 영역은 즉시 렌더링.

### 오프라인 상태

- 캐시된 글감과 여정 카드를 표시. 상단에 "오프라인 모드" 배너 표시.
