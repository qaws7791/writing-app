## 목적

홈은 정보 요약보다 오늘의 행동 결정을 돕는 화면이다.

## 진입 경로

- 앱 첫 진입
- 하단 탭의 홈 재진입

## 범위

- 포함: 인사 영역, 오늘의 추천 글감, 이어서 쓰기, 내 콘텐츠 요약
- 제외: 상세 통계 대시보드, 학습 코스 상세, 공개 피드

## 정보 위계

1. 오늘의 행동 유도
2. 이어서 쓰기
3. 저장한 글감 또는 최근 글의 요약 접근

## 주요 액션

- 추천 글감 카드 선택
- 추천 글감 저장
- 이어서 쓰기 CTA 선택
- 저장한 글감/최근 글 목록 이동

## 상태별 UI

### 기본 상태

- 상단에는 차분한 인사와 현재 시간대 감각을 보여준다.
- 추천 글감은 3~4개의 카드로 노출한다.
- 이어서 쓸 초안이 있다면 홈 중단부에 가장 눈에 띄는 카드로 배치한다.

### 초안 없음 상태

- 이어서 쓰기 영역 대신 새 글 시작 CTA 또는 글감 찾기 CTA를 보여준다.

### 추천 없음 상태

- 추천 영역을 비워두지 않고 저장한 글감 또는 글감 찾기 진입 카드로 대체한다.

### 로딩 상태

- 레이아웃을 유지한 채 추천 카드와 이어서 쓰기 카드 스켈레톤을 보여준다.

### 오류 상태

- 전체 실패보다 부분 실패를 우선 가정한다.
- 추천 오류 시 홈 전체를 막지 않고 대체 CTA를 제시한다.

## 문구 톤 가이드

- 행동을 재촉하지 않고 오늘의 시작을 제안하는 문체를 사용한다.
- 예시: "오늘은 이 글감으로 가볍게 시작해볼 수 있습니다."

## 데스크톱 차이

- 추천 카드와 이어서 쓰기 카드를 2열 이상으로 확장할 수 있다.
- 정보량은 늘릴 수 있지만 홈의 1차 행동 우선순위는 유지한다.
