## 목적

여정 상세는 사용자가 특정 여정의 구성을 파악하고, 다음에 진행할 세션을 시작하는 화면이다.

## 진입 경로

- 홈 탭 > 진행 중인 여정 카드 탭
- 홈 탭 > 여정 탭 > 여정 카드 탭
- 나의 여정 > 여정 카드 탭

## 화면 구성

**헤더**

- 뒤로 가기 (←)
- 여정 제목 (줄임)
- 더보기 메뉴 (⋮)

**히어로 이미지**

- 여정 대표 썸네일 이미지 (전체 너비)

**여정 제목**

- 큰 타이포그래피 (예: "내 주장을 설득력 있게")

**여정 설명**

- 이 여정에서 다루는 내용과 기대 성과 (2~4문장)

**전체 진행률 카드**

- "전체 진행률"
- "{완료 수} / {전체 세션 수} 세션"
- "{달성률}% 달성"
- 세션별 미니 프로그레스 인디케이터

**세션 리스트**

- 수직 리스트로 세션을 순서대로 표시
- 각 세션 카드:

  **완료된 세션 (COMPLETED)**
  - 체크 아이콘 (✓)
  - 세션 제목
  - 상호작용 비활성화 (탭 가능하지 않음) 또는 리뷰 가능

  **진행 중인 세션 (IN_PROGRESS)**
  - 현재 스텝 번호 (예: "2")
  - 세션 제목
  - 세션 설명
  - "지금 시작하기 →" 버튼

  **잠긴 세션 (LOCKED)**
  - 잠금 아이콘 (🔒)
  - 세션 제목
  - 탭 비활성화

---

## 상태별 UI

### 처음 시작하는 여정

- 진행률 카드: "0 / N 세션", "0% 달성"
- 첫 번째 세션만 IN_PROGRESS, 나머지 모두 LOCKED

### 여정 완료

- 진행률 카드: "N / N 세션", "100% 달성"
- 완료 축하 메시지 표시 (애니메이션)
- 모든 세션 체크(✓) 상태

### 로딩 상태

- 히어로 이미지와 세션 리스트 스켈레톤
