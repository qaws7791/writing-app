## 목적

글감 찾기 화면은 탐색 피로를 줄이면서도 선택 가능성을 높이는 탐색 엔진 역할을 한다.

## 진입 경로

- 하단 탭의 글감 찾기
- 홈의 관련 CTA

## 범위

- 포함: 검색창, 필터, 주제 칩, 글감 카드 목록
- 제외: 학습 콘텐츠 탐색, 커뮤니티 추천, AI 추천 이유 설명

## 정보 위계

1. 검색과 필터
2. 빠른 주제 탐색
3. 결과 목록 비교

## 주요 액션

- 검색어 입력
- 길이/주제 필터 조정
- 주제 칩 선택
- 글감 저장
- 상세 진입

## 상태별 UI

### 기본 상태

- 상단에 검색창과 필터 버튼을 고정한다.
- 주제 칩은 수평 스크롤로 가볍게 탐색할 수 있게 둔다.
- 카드 목록은 한 번에 빠르게 훑을 수 있는 정보 밀도로 유지한다.

### 검색 결과 없음 상태

- 막다른 느낌을 주지 않도록 추천 주제 칩과 검색어 수정 제안을 함께 보여준다.

### 필터 결과 없음 상태

- 적용 중인 필터가 보이게 하고, 해제 CTA를 분명하게 제시한다.

### 로딩 상태

- 카드 스켈레톤과 함께 검색/필터 컨트롤은 유지한다.

### 오류 상태

- 결과 영역에 재시도와 홈 또는 저장 글감 이동 대안을 보여준다.

## 문구 톤 가이드

- 검색 실패나 결과 없음 문구도 평가적이지 않게 쓴다.
- 예시: "조금 다른 주제로 다시 찾아볼 수 있습니다."

- 카드 목록은 2~3열 그리드로 확장 가능하다.
