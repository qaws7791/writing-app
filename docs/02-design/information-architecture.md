## 네비게이션 구조

바텀 네비게이션은 5개 탭으로 고정한다. 각 탭의 역할은 명확히 분리되며, 기능이 겹치지 않아야 한다.

| 탭        | 역할                         | 핵심 목적             |
| --------- | ---------------------------- | --------------------- |
| 홈        | 오늘 무엇을 할지 결정하는 곳 | 행동 유도, 루틴 형성  |
| 글감 찾기 | 무엇에 대해 쓸지 발견하는 곳 | 글쓰기 진입 장벽 해소 |
| 학습 활동 | 짧은 훈련을 하는 곳          | 에세이 역량 성장      |
| 글쓰기    | 실제로 쓰는 곳               | 에세이 작성 및 완성   |
| 프로필    | 내 기록을 돌아보는 곳        | 회고 및 성장 확인     |

## 화면별 상세 구조

### 인증

```
인증
├── 로그인
│   ├── 이메일 + 비밀번호
│   └── 소셜 로그인 (Kakao)
└── 회원가입
    ├── 이메일 + 비밀번호
    └── 닉네임 설정
```

---

### 홈

홈은 대시보드가 아닌 **오늘의 행동 유도 화면**이다. 사용자가 앱을 열었을 때 "무엇을 해야 하지?"를 고민하지 않도록 한다.

```
홈
├── 상단 인사 영역
│   └── 개인화 인삿말 (닉네임, 시간대 기반)
├── 오늘의 영감 (글감 카드 4개)
│   └── → 글감 상세 페이지로 이동
├── 이어서 하기
│   └── 최근 작성 중인 에세이
└── 내 콘텐츠 (탭 구조)
    ├── 내가 쓴 글 탭
    │   └── 글 목록 (제목, 내용 미리보기, 마지막 수정일, 글자 수)
    └── 저장한 글감 탭
        └── 글감 목록 (글감 제목, 주제, 난이도)
```

### 글감 찾기

글쓰기 진입 엔진 역할을 한다.

```
글감 찾기
├── 상단 영역
│   ├── 검색창
│   ├── 필터 버튼
│   │   ├── 길이 (짧음 / 보통 / 깊이)
│   │   └── 주제군 (일상 / 사회 / 관계 / 기술 / 자기이해 / 진로)
│   └── 관심 주제 칩 (수평 스크롤)
│
└── 피드 본문
    ├── 검색 결과 글감
    └── 저장한 글감
```

#### 글감 카드 타입

```
글감 카드
├── 한 문장 프롬프트
├── 예상 글 길이 (짧음 / 보통 / 깊이)
├── 추천 레벨
└── 감정 태그 또는 사고 태그
```

#### 글감 상세 페이지

```
글감 상세
├── 글감 내용 (프롬프트, 아웃라인)
├── 관련 태그
├── CTA: "이 글감으로 바로 쓰기" (primary)
└── 저장 액션
```

### 학습 활동

에세이 역량을 단계적으로 키우는 공간이다. 복잡한 개념을 step-by-step 마이크로 레슨으로 분해하여 제공한다.

```
학습 활동
├── 학습 홈
│   ├── 진행 중인 코스
│   ├── 추천 코스
│   └── 완료한 코스
│
├── 코스 목록
│
└── 코스 상세
    ├── 코스 개요 (목표, 소요 시간, 레슨 수)
    └── 레슨 목록
```

### 글쓰기

플랫폼의 심장이다. 자유로운 에디터이지만, 완전한 빈 화면 대신 **가벼운 보조 구조**를 제공한다.

```
글쓰기
├── 글 목록 페이지
│   ├── 헤더: "내 글"
│   ├── 새 글 쓰기 CTA 카드
│   └── 글 목록 (무한 스크롤)
│
└── 에디터 페이지
    ├── 플로팅 헤더
    │   ├── 뒤로가기 아이콘 버튼 (좌)
    │   └── 설정 아이콘 버튼 (우)
    │
    ├── 컨텐츠 영역
    │   ├── 제목 입력 (h1 + contenteditable)
    │   └── 본문 에디터 (Tiptap)
    │
    └── 하단 플로팅 툴바 (기본 상태)
```

#### 에디터 — AI 기능 흐름

```
AI 기능 (AI 버튼 클릭)
│
├── [문장 선택 O] → 문장 교정 모드
│   ├── 1. 툴바 확장 표시
│   └── 2. 기능 선택 시 → 스플릿 뷰 전환
│
└── [문장 선택 X] → 문서 피드백 모드
    ├── 1. 툴바 확장 표시
    └── 2. 기능 선택 시 → 검토 모드 전환
```

### 프로필

개인 기록을 돌아보고 성장을 확인하는 공간이다.

```
프로필
├── 프로필 헤더
│
├── 활동 지표
│
├── 배지 & 마일스톤
│
│
└── 설정
```
