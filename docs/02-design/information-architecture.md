## 글로벌 네비게이션

앱은 하단 탭 바(Bottom Tab Bar) 4개로 주요 섹션을 제공한다.

```
[ 홈 ]  [ 나의 여정 ]  [ 서재 ]  [ 프로필 ]
```

| 탭        | 아이콘    | 역할                              |
| --------- | --------- | --------------------------------- |
| 홈        | house     | 글감 탐색, 여정 탐색, 오늘의 추천 |
| 나의 여정 | book-open | 진행 중인 여정, 여정 찾기         |
| 서재      | pencil    | 내가 쓴 글 목록, 검색             |
| 프로필    | person    | 계정, 설정, 통계                  |

세션 진행 중(전체 화면 플로우)에는 탭 바를 숨긴다.

## 전체 화면 구조

```
앱
├── 홈 탭 (3-tab: 홈 / 글감 / 여정)
│   ├── 홈 서브탭
│   │   ├── 인사 영역
│   │   ├── 오늘의 추천 글감 카드
│   │   └── 진행 중인 여정 카드 목록
│   ├── 글감 서브탭
│   │   ├── 필터 칩 (전체 / 감각 / 회고 / 의견)
│   │   └── 글감 카드 리스트
│   └── 여정 서브탭
│       ├── 필터 칩 (전체 / 글쓰기 기술 / 마음챙김 / 기술)
│       └── 여정 카드 리스트
│
├── 글감 상세 (push)
│   └── → 글 에디터 (글쓰기 시작)
│
├── 나의 여정 탭
│   ├── 진행 중인 여정 섹션 (카드 목록)
│   └── 여정 찾기 섹션 (카테고리 그리드)
│
├── 여정 상세 (push)
│   └── → 세션 진행 플로우 (전체 화면)
│       ├── 스텝: 개념 학습 (Learn)
│       ├── 스텝: 예시 읽기 (Read)
│       ├── 스텝: 가이드 질문 (Guided Question)
│       ├── 스텝: 글쓰기 (Write)
│       ├── 스텝: AI 피드백 (Feedback)
│       └── 스텝: 수정 (Revise)
│
├── 서재 탭
│   ├── 검색 바
│   └── 글 카드 목록
│       └── → 글 상세 (읽기 뷰)
│           └── Writing Prompt 접힘 카드
│               └── → 글감 상세
│
└── 프로필 탭
    ├── 프로필 영역
    ├── 통계 카드
    ├── 계정 관리
    ├── 접근성 설정
    └── 서비스 및 데이터
```

## 핵심 개념 모델

### 여정-세션-스텝 계층 구조

```
여정 (Journey)
├── 세션 1 (Session) — 10~20분 완결형 학습 노드
│   ├── 스텝 1 — 개념 학습 (Learn)
│   ├── 스텝 2 — 가이드 질문 (Guided Question)
│   ├── 스텝 3 — 글쓰기 (Write)
│   └── 스텝 4 — AI 피드백 (Feedback)
├── 세션 2 (Session)
│   └── ...
└── 세션 N — 최종 에세이 작성 + 종합 피드백
```

### 글감-글 관계

```
글감 (Writing Prompt)
├── 유형: 감각 | 회고 | 의견
├── 응답 수 (responseCount)
└── 글 목록 (Writing[])
    └── 글 (Writing)
        ├── 제목, 본문, 단어 수
        ├── 버전 이력 (WritingVersion[])
        └── AI 피드백 (aiFeedback, JSON)
```

## 엔티티 정의

| 엔티티                    | 정의                                          | 예시                                                          |
| ------------------------- | --------------------------------------------- | ------------------------------------------------------------- |
| **글감 (Writing Prompt)** | 사용자의 사고를 촉발하는 질문·주제·상황 카드  | "먼지 쌓인 서랍을 열었을 때, 당신의 시선을 멈추게 한 물건은?" |
| **여정 (Journey)**        | 특정 역량을 목표로 한 구조화된 커리큘럼       | "내 주장을 설득력 있게" (7세션, 논증 에세이)                  |
| **세션 (Session)**        | 여정 내 하나의 학습 노드                      | "근거의 구조화" — 반론 대비 연습                              |
| **스텝 (Step)**           | 세션 내 하나의 마이크로 활동 단위             | 개념 카드 읽기 / 가이드 질문 답변 / 글쓰기 / 피드백 확인      |
| **글 (Writing)**          | 사용자가 작성한 텍스트. 글감 또는 세션에 연결 | 에세이 초안, 수정본                                           |
| **서재 (Library)**        | 사용자가 작성한 모든 글의 아카이브            | —                                                             |

## 글감 유형 분류

| 유형                  | 설명                                    | 메타인지 목표          |
| --------------------- | --------------------------------------- | ---------------------- |
| **감각 (Sensory)**    | 감각적 경험에서 출발하는 자유 글쓰기    | 관찰력, 표현력         |
| **회고 (Reflection)** | 과거 경험을 되돌아보는 성찰적 글쓰기    | 자기 인식, 의미 구성   |
| **의견 (Opinion)**    | 특정 주제에 대한 입장을 논증하는 글쓰기 | 논리적 사고, 근거 제시 |

## 여정 카테고리

| 카테고리         | 설명                              | 예시 여정                          |
| ---------------- | --------------------------------- | ---------------------------------- |
| **글쓰기 기술**  | 에세이 구조, 문단 구성, 퇴고 전략 | "첫 문장의 힘", "설득하는 글쓰기"  |
| **마음챙김**     | 감정 탐색, 자기 성찰, 일상 관찰   | "새벽의 대화", "나를 찾는 여행"    |
| **기술 (Skill)** | 특정 직무·상황 맞춤형 글쓰기      | "업무 이메일 글쓰기", "자기소개서" |

## 세션 상태 전이

```
LOCKED → IN_PROGRESS → COMPLETED
   ↑
이전 세션 COMPLETED 시 자동 전환
```

- **LOCKED:** 잠금 아이콘(🔒) 표시, 탭 비활성화
- **IN_PROGRESS:** 현재 스텝 번호 표시, "지금 시작하기 →" 버튼 활성화
- **COMPLETED:** 체크(✓) 아이콘 표시

## 네비게이션 뎁스

| 뎁스 | 예시                                   |
| ---- | -------------------------------------- |
| 0    | 탭 바 (홈 / 나의 여정 / 서재 / 프로필) |
| 1    | 글감 상세, 여정 상세, 글 상세          |
| 2    | 세션 진행 플로우 (전체 화면 오버레이)  |

전체 화면 오버레이(세션 플로우)는 탭 바를 덮는다. 뒤로 가기로 여정 상세로 복귀한다.
