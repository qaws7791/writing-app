## 목적

게이미피케이션은 사용자가 글쓰기와 여정 진행을 지속하도록 내재적·외재적 동기를 강화한다. 단, 진행률과 보상이 "글쓰기 과정을 통한 사고력 훈련"이라는 핵심 가치를 방해하지 않도록 설계한다.

## 리텐션 메커니즘

| 메커니즘           | 설명                                 | 구현 우선순위 |
| ------------------ | ------------------------------------ | ------------- |
| **여정 진행률**    | 세션별/여정별 프로그레스 바와 달성률 | P0 (MVP)      |
| **세션 달성 보상** | 세션 완료 시 시각적 축하 애니메이션  | P0 (MVP)      |
| **글쓰기 스트릭**  | 연속 글쓰기 일수 추적                | P1            |
| **단어 수 누적**   | 총 작성 단어 수 카운터               | P1            |
| **여정 완료 뱃지** | 여정 완료 시 프로필에 뱃지 추가      | P2            |

## 동기 부여 전략

**내재적 동기 (Intrinsic)**

- AI 피드백을 통한 실력 향상 체감
- 자신의 글 아카이브 축적 (서재)
- 메타인지 질문을 통한 지적 자극

**외재적 동기 (Extrinsic)**

- 여정 진행률과 달성률 시각화
- 스트릭 시각화
- 글감 응답 수를 통한 사회적 참여감 ("지금까지 {N}명이 응답했어요")

## 진행률 시각화

### 여정 카드 (홈 / 나의 여정)

- 프로그레스 바 + 퍼센트 (예: "65%")
- 부제 표시 (예: "완성된 단편선", "자아 성찰 에세이")

### 여정 상세

- "전체 진행률: {완료 수} / {전체 세션 수} 세션 / {달성률}% 달성"
- 세션별 미니 프로그레스 인디케이터

### 세션 진행 플로우

- 상단 프로그레스 바: 현재 스텝 / 전체 스텝 수
- REQ-SES-001

## 세션 완료 보상

- 완료 시 시각적 축하 애니메이션 표시
- "세션 완료!" 메시지
- 다음 세션 잠금 해제 안내
- 여정 상세의 진행률 업데이트

## 여정 완료 (P2)

- 여정 완료 축하 화면 (전체 화면 애니메이션)
- 프로필에 여정 완료 뱃지 추가
- 여정 상태 변경 (예: "완성된 단편선")
- REQ-JRN-006

## 설계 주의사항

- 게이미피케이션 요소가 글쓰기 자체보다 더 중요하게 보이지 않도록 한다.
- 스트릭이 끊겼을 때 죄책감보다 재참여 유도로 설계한다 (예: "다시 시작해봐요")
- 분량 채우기보다 피드백-수정 반복에 집중하도록 유도한다.
