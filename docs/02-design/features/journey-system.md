## 목적

여정은 특정 글쓰기 역량을 목표로 한 구조화된 커리큘럼이다. 여정 시스템은 사용자가 단계별로 성장하는 경험을 제공하고, 진행률 시각화로 리텐션을 확보한다.

## 사용자 문제

- 체계적 학습 기회 부재 → 에세이 구조/퇴고 전략을 혼자 습득 불가
- 실력이 느는지 알 수 없음 → 동기 소실

## 여정 카테고리

| 카테고리         | 설명                         | 예시                               |
| ---------------- | ---------------------------- | ---------------------------------- |
| **글쓰기 기술**  | 에세이 구조, 문단, 퇴고 전략 | "첫 문장의 힘", "설득하는 글쓰기"  |
| **마음챙김**     | 감정 탐색, 자기 성찰         | "새벽의 대화", "나를 찾는 여행"    |
| **기술 (Skill)** | 직무·상황 맞춤 글쓰기        | "업무 이메일 글쓰기", "자기소개서" |

## 핵심 기능

### 1. 여정 탐색

- 홈 탭 > 여정 서브탭에서 카드 리스트로 표시
- 필터 칩으로 카테고리별 필터링 (전체 / 글쓰기 기술 / 마음챙김 / 기술)
- REQ-JRN-001

### 2. 여정 상세

- 여정 카드 탭 시 여정 상세 화면 이동
- 히어로 이미지, 여정 설명, 전체 진행률, 세션 리스트 표시
- REQ-JRN-002

### 3. 세션 선형 진행 구조

- 세션은 순서대로 배치. 이전 세션이 COMPLETED여야 다음 세션이 IN_PROGRESS로 전환
- 대기 중(LOCKED) 세션: 잠금(🔒) 아이콘, 탭 비활성화
- REQ-JRN-003, REQ-JRN-004

### 4. 세션 시작

- "지금 시작하기 →" 버튼 탭 시 해당 세션의 첫 스텝을 전체 화면 플로우로 시작
- REQ-JRN-005

### 5. 여정 완료

- 마지막 세션 완료 시 여정 완료 축하 화면 표시
- 여정 상태가 "완료된 단편선" 등으로 변경
- 완료 뱃지 프로필에 추가 (P2)
- REQ-JRN-006

## 세션 상태 전이

```
LOCKED → IN_PROGRESS → COMPLETED
```

- **LOCKED → IN_PROGRESS:** 이전 세션이 COMPLETED일 때 자동 전환
- **IN_PROGRESS → COMPLETED:** 세션의 마지막 스텝 완료 시

## 진행률 표시

- **여정 카드:** 프로그레스 바 + 퍼센트 (예: "65%")
- **여정 상세:** "{완료 수} / {전체 세션 수} 세션 / {달성률}% 달성"
- **홈 탭:** 진행 중인 여정 카드에 진행률 표시

## 예외 상태

- 여정 데이터 로딩 실패: 이전 캐시 표시 + 새로고침 버튼
- 세션 잠금 해제 실패 (서버 오류): 로컬에서 시뮬레이션 후 동기화
