## 목적

글감은 사용자의 사고를 촉발하는 질문·주제·상황 카드다. v2.1에서 글감은 Tertiary 기능으로, 독립 탭이나 상세 화면 없이 글쓰기 진입 화면과 에디터 내 Bottom Sheet에서만 노출된다.

> **v1과의 차이:** 홈 화면 내 글감 서브탭, 글감 상세 화면, 글감별 글 목록이 모두 제거되었다. 글감은 글쓰기 시작을 돕는 보조 도구로만 존재한다.

## 사용자 문제

- "무엇을 써야 할지 모르겠다" — 소재/주제 생성 어려움
- 빈 화면 앞의 막막함 → 글쓰기 시작 회피

## 글감 유형

| 유형                  | 설명                 | 예시                                                          |
| --------------------- | -------------------- | ------------------------------------------------------------- |
| **감각 (Sensory)**    | 감각적 경험에서 출발 | "창문을 열었을 때 처음 느낀 냄새는?"                          |
| **회고 (Reflection)** | 과거 경험을 성찰     | "먼지 쌓인 서랍을 열었을 때, 당신의 시선을 멈추게 한 물건은?" |
| **의견 (Opinion)**    | 특정 주제 입장 논증  | "SNS에서 익명성은 자유인가, 방종인가?"                        |

## 핵심 기능

### 1. 글쓰기 진입 화면에서 글감 탐색

- 글쓰기 탭 > "새 글 쓰기" 탭 → "오늘 어떤 글을 써볼까요?" 화면
- 필터 칩으로 유형별 필터링 (전체 / 감각 / 회고 / 의견)
- 글감 카드에 유형 태그, 제목, 본문 요약, 응답 수 소형 표시
- 글감 카드 탭 → 해당 글감이 참조 카드로 고정된 에디터 열림

### 2. 에디터 내 Bottom Sheet에서 글감 추가 탐색

- 빈 에디터에서 "아이디어가 필요하신가요?" 버튼 탭 → Bottom Sheet
- 필터 칩 + 글감 카드 리스트 (진입 화면과 동일 구성)
- 글감 카드 탭 → 시트 닫힘 → 에디터 상단에 글감 참조 카드 고정

### 3. 에디터 내 글감 참조 카드

- 글감을 선택한 경우 에디터 상단에 접힘/펼침 카드로 표시
- 글감 제목과 본문 확인 가능
- 글쓰기 영역 확보를 위해 기본 접힘 상태 유지

### 4. 응답 수 (Social Proof)

- 각 글감 카드에 "N명이 이 글감으로 글을 썼어요" 소형 표시
- 심리적 안전감을 주는 사회적 증거 역할에 한정
- 글감 상세 화면으로 연결되지 않음
