## 목적

글감은 사용자의 사고를 촉발하는 질문·주제·상황 카드다. 글감 시스템은 사용자가 오늘 쓸 주제를 빠르게 발견하고, 글쓰기를 시작하는 심리적 장벽을 낮추는 역할을 한다.

## 사용자 문제

- "무엇을 써야 할지 모르겠다" — 소재/주제 생성 어려움
- 빈 화면 앞의 막막함 → 글쓰기 시작 회피

## 글감 유형

| 유형                  | 설명                 | 예시                                                          |
| --------------------- | -------------------- | ------------------------------------------------------------- |
| **감각 (Sensory)**    | 감각적 경험에서 출발 | "창문을 열었을 때 처음 느낀 냄새는?"                          |
| **회고 (Reflection)** | 과거 경험을 성찰     | "먼지 쌓인 서랍을 열었을 때, 당신의 시선을 멈추게 한 물건은?" |
| **의견 (Opinion)**    | 특정 주제 입장 논증  | "SNS에서 익명성은 자유인가, 방종인가?"                        |

## 핵심 기능

### 1. 오늘의 추천 글감

- 매일 오전 6시에 자동 갱신
- 사용자의 관심사와 글쓰기 수준 기반 선정
- 홈 탭 > 홈 서브탭에서 강조 카드로 표시
- REQ-HOME-002

### 2. 글감 목록 탐색

- 홈 탭 > 글감 서브탭에서 카드 리스트로 표시
- 필터 칩으로 유형별 필터링 (전체 / 감각 / 회고 / 의견)
- 각 카드에 응답 수 실시간 표시 (REQ-PRM-005)
- REQ-PRM-001

### 3. 글감 상세

- 카드 탭 시 글감 상세 화면 이동
- 글감 제목, 본문, 참여 통계 (참여 작가 수 / 작성된 수필 수) 표시
- 해당 글감에 대한 다른 사용자의 글 목록 표시
- REQ-PRM-002

### 4. 글쓰기 시작

- "글쓰기 시작" 버튼 탭 시 해당 글감이 연결된 글 에디터 열기
- Writing Prompt 정보를 에디터 상단에 참조용으로 표시 (접힘 상태)
- REQ-PRM-003

### 5. 북마크

- 글감 카드의 북마크 아이콘(🔖) 탭 시 북마크 목록에 추가/제거
- 별도 토스트 없이 아이콘 상태 변경으로 피드백
- REQ-PRM-004

## 상호작용 규칙

- 글감 카드에서 탭 가능한 영역: 카드 전체 (북마크 버튼 제외)
- 북마크 버튼: 독립 탭 영역 (카드 탭과 별개)
- 응답 수는 실시간으로 업데이트 (폴링 또는 서버-센트 이벤트)

## 예외 상태

- 글감 목록 로딩 실패: 캐시된 마지막 목록 표시 + 새로고침 버튼
- 글감 상세 로딩 실패: "다시 시도해볼까요?" + 재시도 버튼
