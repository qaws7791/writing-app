## 문서 역할

이 폴더는 user-flow를 실제 상호작용 단위로 분해한 중간 설계 문서를 관리한다.

각 feature 문서는 목적, 사용자 문제, 진입 조건, 상호작용 규칙, 예외 상태, 연결된 flow와 screen을 함께 정의한다.
