## 원칙

AI는 글을 대신 쓰는 기능이 아니라 사용자의 생각과 문장을 다듬는 보조 기능입니다.

## 구조

### Layer 1. 선택 영역 보조

- 사용자가 문장이나 단어를 선택한 뒤 AI 도구를 연다.
- 어휘 개선, 문장 명료화, 호흡 조정 같은 제한된 제안을 제공한다.
- 자동 적용은 하지 않고 사용자가 선택한다.

### Layer 2. 전체 문서 검토

- 작성 이후 전체 문서를 대상으로 맞춤법, 중복 표현, 문단 연결 흐름을 점검한다.
- 수정 제안은 인라인으로 확인하고 항목별로 수락 또는 거절한다.

## 포함할 기능

- 표현 제안
- 모호성 지적
- 문장 길이와 호흡 조정
- 맞춤법과 띄어쓰기 검토
- 문단 연결 피드백

## 포함하지 않을 기능

- 전체 초안 자동 생성
- 논지 자동 완성
- 사용자 목소리를 대체하는 재작성

## 제품적 의미

AI 기능의 성공 기준은 사용자가 더 빨리 답을 받는 것이 아니라, 더 잘 쓰고 더 잘 고치게 되는 것입니다.
