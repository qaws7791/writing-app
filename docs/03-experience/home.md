# Home

## 목적

홈은 사용자가 앱을 열었을 때 바로 다음 행동을 결정하게 돕는 화면입니다. 분석용 대시보드보다 행동 유도가 우선입니다.

## 핵심 구성

- 인사와 현재 상태
- 오늘의 추천 활동
- 이어서 작성할 초안
- 저장한 글감 또는 최근 글

## UX 원칙

- 오늘 해야 할 일을 하나 또는 소수로 압축해 제시한다.
- 최근 행동과 관심사를 반영해 개인화한다.
- 카드 중심 구조로 가볍게 훑고 바로 진입할 수 있게 한다.

## 제외할 것

- 지나치게 많은 통계
- 홈에서 모든 기능을 한 번에 노출하는 구조
