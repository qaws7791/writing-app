# Prompt Discovery

## 목적

글감 찾기는 사용자가 카테고리 목록을 탐색하는 기능이 아니라, 지금 쓰고 싶은 문제의식과 감정을 발견하는 흐름이어야 합니다.

## 화면 원칙

- 피드형 구조를 기본으로 한다.
- 주제, 기분, 상황, 난이도를 기준으로 좁혀 볼 수 있어야 한다.
- 저장과 나중에 쓰기 행동이 간단해야 한다.

## 구성 요소

- 추천 글감 피드
- 필터 칩과 바텀시트
- 저장 목록 진입
- 글감에서 바로 학습 활동 또는 초안 시작

## 성공 기준

- 사용자가 글감을 고르는 데 오래 머무르지 않는다.
- 탐색 결과가 실제 초안 시작으로 이어진다.
