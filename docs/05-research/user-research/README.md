## 핵심 인사이트 요약

### AI 글쓰기 고충 6대 축

1. 사실·인용 신뢰성 (환각, 가짜 참고문헌)
2. 학습·사고 (비판적 사고 외주화, 자기 효능감 저하)
3. 스타일·창의성 (개인 문체 소실, 균질화)
4. 워크플로·프롬프트 (반복 시도 비용, 통합 어려움)
5. 윤리·정책 (표절 리스크, 탐지 불안정)
6. 프라이버시 (연구 데이터·개인정보 통제 불가)

### 시장 미충족 니즈

- 생산성 도구(AI 대필)와 학습 도구(역량 훈련) 목표 불일치
- 피드백 깊이 부족 (문법 수준에서 고차원 전략 미흡)
- 장기 커리큘럼과 진행감 부재
- 자기주도 온보딩 부재
- 동기·습관 설계 미흡

### 글필의 포지셔닝 근거

에세이 리비전 루프에 집중하는 개인용 코치: AI 생성이 아닌 AI 코칭 중심.
