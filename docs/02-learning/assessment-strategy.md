# Assessment Strategy

## 평가 목적

평가는 점수화보다 사용자가 자신의 글쓰기 습관과 성장 방향을 이해하도록 돕는 데 목적이 있습니다.

## 평가 기준

- 시작 빈도: 얼마나 자주 글쓰기를 시작했는가
- 완성 흐름: 초고에서 퇴고까지 이어지는가
- 구조화: 문단과 논지 전개가 나아지는가
- 표현 명료성: 모호함과 중복이 줄어드는가
- 회고 습관: 작성 후 스스로 진단하는가

## 피해야 할 것

- 한 번의 결과물만 보고 사용자를 평가하는 방식
- AI가 생성한 점수에 과도하게 의존하는 방식
- 공개 반응 수를 글쓰기 성과처럼 사용하는 방식
