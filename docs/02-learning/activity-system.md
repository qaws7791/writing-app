# Activity System

## 활동 유형

- `워밍업`: 키워드 확장, 관찰 기록, 짧은 문장 만들기
- `구조 훈련`: 서론, 본론, 결론 또는 문단 전개 연습
- `표현 훈련`: 모호한 문장 명료화, 문장 호흡 조정
- `퇴고 훈련`: 삭제, 재배열, 중복 표현 줄이기
- `회고 훈련`: 내가 막힌 지점과 다음 시도 기록하기

## 활동 설계 규칙

- 5분에서 15분 안에 끝날 수 있어야 한다.
- 결과는 바로 초고 작성으로 이어질 수 있어야 한다.
- 성취보다 반복과 전이를 우선한다.

## 제품 연결 방식

- 홈은 오늘의 추천 활동을 제시한다.
- 글감 찾기에서 선택한 주제는 활동 추천에 반영된다.
- 활동 결과는 에디터 초안과 연결된다.
