# Glossary

## 제품 용어

- `essay`: 사용자가 한 편의 완결된 글로 작성하는 기본 산출물
- `draft`: 아직 공개되지 않은 작업 중인 글
- `revision`: 초고 이후 구조, 문장, 표현을 다시 다듬는 단계
- `publication`: 글을 다른 사용자에게 공개하는 행위
- `prompt discovery`: 글감을 찾고 선택하는 탐색 경험

## 학습 용어

- `learning activity`: 짧은 단위의 훈련 과제
- `scaffolding`: 사용자가 스스로 쓰도록 돕는 단계적 안내
- `reflection`: 작성 이후 회고와 자기 피드백

## AI 용어

- `ai assistant`: 사용자의 글쓰기를 보조하지만 대신 써주지 않는 기능 집합
- `coaching`: 질문, 관찰, 피드백으로 사고를 돕는 방식
- `correction`: 맞춤법, 표현, 연결성 등 교정 중심 보조

## 문서 용어

- `canonical document`: 현재 기준이 되는 문서
- `reference material`: 판단 근거가 되는 리서치 또는 외부 자료
- `archive`: 실험안, 폐기안, 대체된 문서
