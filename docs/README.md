# Writing App Documentation

이 디렉토리는 글쓰기 플랫폼의 기준 문서 허브입니다. 제품 방향, 학습 설계, UX, 엔지니어링, 운영, 리서치 자료를 한곳에서 관리합니다.

## 읽는 순서

1. `00-governance`: 문서 규칙과 용어
2. `01-product`: 제품 목표와 문제 정의
3. `02-learning`: 글쓰기 학습 모델
4. `03-experience`: 화면 구조와 상호작용 원칙
5. `04-engineering`: 모노레포 구조와 기술 문서
6. `05-operations`: 보안, 운영, 릴리스 대응
7. `06-research`: 사용자 리서치와 참고자료
8. `07-prompts`: AI 활용 프롬프트 문서
9. `99-archive`: 실험 결과물과 대체된 문서

## 문서 분류

- `canonical`: 현재 기준 문서. 의사결정과 구현은 이 문서를 우선한다.
- `reference`: 리서치, 참고자료, 외부 근거.
- `archive`: 더 이상 기준이 아니지만 보존이 필요한 문서.

## 현재 상태

- 번호가 붙은 디렉토리가 현재 기준 체계입니다.
- 루트의 `legacy-*` 파일과 디렉토리는 기존 초안 원본이며, 내용은 `99-archive`와 새 기준 문서에 반영해 나갑니다.
- 새 문서를 작성할 때는 구현된 내용과 계획 중인 내용을 구분해서 적습니다.
