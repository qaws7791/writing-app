발견한 문제를 바탕으로 ai를 활용한 자기주도 에세이 교육 플랫폼을 단일 html로 만들어줘(ui와 페이지 이동만 구현하여 기능은 구현하지 않음. 토큰을 적게 사용하는 방향으로 작성하세요.) mvp는 핵심 기능 글쓰기와 글배우기 2가지만 설계하며, 커뮤니티 기능이나 다른 큰 기능은 넣지 않음. 사이트 운영자가 관리해야 하는 컨텐츠: 글감, 커리큘럼 / 사용자가 관리해야하는 컨텐츠: 글.
주요 기능:

글쓰기: 에디터를 사용하여 글 쓰기, ai를 활용하여 글 고치고 피드백 받기, 빈 페이지 두려움을 극복하기 위한 글감을 제공. 글감은 사이트 운영자가 주기적으로 추가. mvp 시작시에는 1000개를 넣고 시작. 사용자는 머릿속에 떠오르지 않는 주제에 대해 글감 찾기에서 탐색하면서 이에 대해 떠오르게 됨.
글쓰는 방법 배우기: 글쓰기 능력 향상을 위한 다양한 커리큘럼 제공. 모든 커리큘럼은 인터랙티브한 경험을 제공한다. 사이트 운영자는 커리큘럼을 신뢰할 수 있는 출처에 기반하여 설계한다
핵심 가치

플랫폼이 현실적으로 꼭 모든 문제를 해결할 필요는 없다.
플랫폼의 성공 가치와 사용자의 만족도를 중점으로 기획하라.
글을 쓰기 시작하는 초등학생이나 글쓰기를 해본 적 없는 노인도 글쓰기를 배울 수 있도록 처음부터 인도하는 플랫폼으로 기획하라.
레퍼런스 서비스: Brilliant, duolingo,Write & Improve with Cambridge,Packback Deep Dives
플랫폼 방향:

이 플랫폼은 책이나 강의를 대체하지 않는다.
이 플랫폼은 노션, 옵시디언 같은 생산성 기록 앱을 대체하지 않는다
이 플랫폼은 브런치, 문피아 같은 출판 플랫폼을 대체하지 않는다.
이 플랫폼은 일기 앱을 대체하지 않는다.
이 플랫폼은 감정 치유 앱을 대체하지 않는다.
디자인 방향:
Meta Threads의 UI 테마:
Threads는 "KISS(Keep It Simple Stupid)" 원칙을 UI 전반에 적용합니다. 과도한 색상, 그래픽 요소 없이 텍스트와 인터랙션 자체가 중심이 되도록 설계되어 있으며, Instagram의 시각 언어를 계승하면서도 더 텍스트 중심적인 경험을 제공합니다.
Threads의 색상 시스템은 **유연성, 응집력, 구성 가능성(configurability)**을 목표로 리뉴얼됐습니다. 주요 특징은 다음과 같습니다.

모노크롬 팔레트: 강조색 없이 흑/백/회색 계열 위주로 구성
대비 정규화: 값들 간 대비를 균일하게 맞춰 접근성 조합의 수를 확장
기존 팔레트 확장: 기존 색상 값의 범위를 넓혀 더 많은 활용 케이스 지원

라이트/다크 모드 모두 지원하며, 별도 앱 설정이 아닌 디바이스 시스템 테마를 따라감.
