## 왜 초기부터 필요한가

글쓰기 서비스는 창작과 자기노출이 함께 일어나므로, 운영 품질이 사용자 유지에 직접적인 영향을 줍니다. 모더레이션과 고객지원은 후순위 기능이 아니라 생존 조건입니다.

## 주요 이슈

- 공격적 댓글과 괴롭힘
- 신고 접수 후 검토 지연
- 공개 범위 오해로 인한 불만
- 유료 기능 또는 결제 이슈 대응
- 초안 유실과 데이터 신뢰 문제

## MVP 운영 기준

- 짧고 명확한 커뮤니티 규칙
- 신고, 검토, 조치, 이의제기 흐름
- 제한적인 피드백 노출 정책에서 시작
- 사용자 문의와 버그 신고 채널 분리

## 운영 지표

- 신고 접수 대비 처리 시간
- 반복 위반자 비율
- 공개 후 삭제 또는 비공개 전환 비율
- 고객지원 최초 응답 시간
