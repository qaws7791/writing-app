# Web

## 현재 상태

웹 앱은 `apps/web`에 위치하며 Next.js 16과 React 19를 사용합니다. 편집 경험을 위해 TipTap 관련 의존성이 포함되어 있고, 공용 UI는 `@workspace/ui`에서 가져옵니다.

## 책임

- 제품 UI 렌더링
- 글쓰기, 탐색, 학습, 회고 흐름 제공
- API와 사용자 사이의 주 인터페이스

## 구현 원칙

- 모바일 우선
- 비주얼보다 읽기와 쓰기 흐름 우선
- 공용 UI 패키지를 통한 일관성 유지

## 문서 연계

- 화면 구조는 `03-experience` 문서와 함께 봅니다.
- 컴포넌트 수준 규칙은 `ui-package.md`와 Storybook 문서를 참고합니다.
