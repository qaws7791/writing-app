## 방향

이 플랫폼의 커리큘럼은 완성된 이론 전달보다 실제 작성 행동을 늘리는 쪽에 초점을 둡니다.

## 원칙

- 짧고 반복 가능한 단위로 훈련한다.
- 설명보다 예시와 직접 수행을 우선한다.
- 초고 작성과 퇴고를 별도 기술로 다룬다.
- 사용자의 주제와 경험을 기반으로 훈련한다.
- 정답형 평가보다 표현력과 구조화 능력의 향상을 본다.

## 다루는 핵심 역량

- 글감 발견
- 논지 세우기
- 문단 구성
- 사례와 근거 배치
- 문장 명료화
- 퇴고와 회고
