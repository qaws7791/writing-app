### 설계 원칙

| 원칙                 | 설명                                                                         |
| -------------------- | ---------------------------------------------------------------------------- |
| **심리적 안전 우선** | 모든 기능은 "쓰기 시작하는 것"을 가장 쉽고 안전하게 만드는 방향으로 설계     |
| **과정 중심**        | 완성된 글보다 글쓰기 과정(아이디어 → 초고 → 퇴고 → 공유)을 가시화하고 지원   |
| **점진적 공개**      | 비공개 → 소그룹 → 전체 공개의 단계적 노출로 평가 불안 최소화                 |
| **증거 기반 교육**   | 마이크로 레슨, 예시글, 스캐폴딩 등 연구로 검증된 교육적 개입을 플랫폼에 내장 |
| **커뮤니티 안전**    | 괴롭힘 방지, 명확한 규칙, 투명한 집행으로 지속 가능한 글쓰기 문화 형성       |
